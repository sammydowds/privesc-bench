# PrivEsc Evals

## Summary

This project was inspired by [CVE-bench from research at UICUC](https://arxiv.org/pdf/2503.17332v3) in agentic red teaming evaluation for specific CVEs. 

I built a few sandboxes containing vectors to esecelate privilege. One very simple case (SUID bit set on `cat`) and a slightly more complicated box which requires hijacking binary path, understanding the cron environment, and writing a file.
